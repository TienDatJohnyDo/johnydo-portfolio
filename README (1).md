# Johny Do — Personal Portfolio

A personal portfolio website built with pure HTML, CSS, and vanilla JavaScript. No frameworks, no build steps — just open `index.html` and go.

## ✨ Features

- Pastel purple & white design with full **dark mode** (remembers your preference)
- Smooth scroll fade-in animations
- Sections: Hero, About, Experience, Education, Projects, Contact
- Fully responsive (mobile-friendly)
- Zero dependencies

## 🚀 Getting Started

```bash
git clone https://github.com/YOUR_USERNAME/johnydo-portfolio.git
cd johnydo-portfolio
open index.html   # or just double-click it
```

## 📝 Customising

All content is in `index.html`. Things to update before going live:

- [ ] Replace `href="#"` social links with your real GitHub, LinkedIn, YouTube URLs
- [ ] Update the email in the contact section (`hello@johnydo.com`)
- [ ] Add your real project links (live demos + GitHub repos)
- [ ] Swap in a profile photo if you'd like one

## 🌐 Deploying to Vercel

Since you're already on Vercel:

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → **New Project** → Import your repo
3. Vercel will auto-detect it as a static site — hit **Deploy**

Or via Vercel CLI:
```bash
npm i -g vercel
vercel
```
